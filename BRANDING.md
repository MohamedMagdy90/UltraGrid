## Color Usage Guidelines
### Do's
- ✅ Use Neon Green (#2EFF2E) for primary actions and brand elements
- ✅ Use Black (#000000) as the primary background color for dark mode
- ✅ Use Dark Red (#8B0000) sparingly for emphasis and alerts
- ✅ Ensure sufficient contrast between text and backgrounds
### Don'ts
- ❌ Don't use Neon Green for large text blocks (hard to read)
- ❌ Don't combine Dark Red with Neon Green in close proximity
- ❌ Don't use colors at reduced opacity that compromise accessibility

## Logo Usage
### Text Logo Format
```
UltraBoost
```
When displaying the product name:
- Always capitalize "U" and "B" (UltraBoost)
- Never use all caps (ULTRABOOST) except in special contexts
- Never use all lowercase (ultraboost) in marketing materials
### Logo Placement
- Minimum clear space: Equal to the height of the "U" in UltraBoost
- Minimum size: 16px font size for digital, 8pt for print
